java script 
	웹에 움직임을 추가할 떄 사용 
	서버와의 통신
	
	웹의 3대 요소 중 1개 
	HTML + CSS + JS

	왜 js ? 오랜 기간동안 Js를 통해 웹 애플리케이션을 구축하고 유지 관리 해옴, 브라우저에게 명령을 내리는 표준이라 생각


함수 선언
	function 키워드를 사용

	document.getElementById('속성값').style.color 와 같은 문법을 통해 반응을 추가할 수 있다.
		-> 작성 할 떄마다 코드가 길고 복잡해진다는 단점이 발생

	-> JQuery라는 미리 개발자들이 짜둔 코드를 가져와서 사용 

	$('#속성값').html('안녕');

	list = [원소1, 원소2 ... ] 형태로 구성, 인덱싱을 통해 데이터 참조 가능
	text가 아닌 html 코드를 추가하고 싶은 경우, `` (백틱)을 사용하여 추가
	이 경우 .text가 아닌 .append를 통해 추가
	예시 `<p>데이터</p>`

	내용으로 고정 값이 아닌 변수를 넣고 사용하고 싶은 경우
	${변수명}을 사용
	예시 `<p>${tmpText}</p>` 

fetch
	'fetch'는 인터넷을 통해 데이터를 요청하고 받아오는 과정
	짧은 코드로 데이터를 가져올 수 있다.

span tag
	<span></span>일부 문자를 묶을 때 사용
		=> id 또한 부여할 수 있어
		=> 문자열 중 일부만 바꾸고 싶을 경우 사용하면 좋다.
